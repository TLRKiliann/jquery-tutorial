# jquery-tutorial
jQuery with Vite &amp; TS

## Install

└─ $ ▶ pnpm create vite

└─ $ ▶ cd my-project

└─ $ ▶ pnpm install

└─ $ ▶ pnpm run dev

└─ $ ▶ pnpm install --save-dev jquery

**jQuery & App.tsx**

```
$(() => {
    setTimeout(() => {
        alert('jQuery triggered via app.js')
    }, 2500);
});
```
